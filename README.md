# 台大經濟系研究所學門歸類工具

協助台大經濟系碩士生輸入修過的課程，自動對應所屬學門，並找出所有符合畢業門檻（2 個不同學門、每學門至少 2 科）的組合。

**[>> 開啟工具](https://liaojyu.github.io/econ-field-planner/)**

## 使用方式

1. 在課號或課名欄位輸入，從下拉清單選取課程
2. 點擊「學門歸類分析」查看所有合法組合
3. 選用偏好組合後，可下載 CSV 或複製文字

## 課程資料版本

依據系上學門分類表，最後更新日期：**2025 年 12 月 23 日**

如需更新課程資料，只需修改 `courses.json`，不需動到 `index.html`。

## 學門清單（共 13 個）

總體與貨幣、經濟發展與政策、國際貿易與區域經濟、國際金融、環境健康經濟、人力資源、公共經濟、產業組織、計量經濟、法律經濟、財務經濟、經濟史與方法論、賽局與數理經濟
